Inveskit: 매매기록 기반 투자 조언 서비스

Inveskit은 개인 투자자가 자신의 매매 기록을 바탕으로 인공지능 언어 모델(LLM)의 맞춤형 투자 조언을 받는 웹 서비스입니다. 파편화된 정보를 넘어서, 개인화된 데이터 기반의 합리적인 투자 결정을 돕는 것을 목표로 합니다.

주요 기능

매매 기록 등록: 사용자가 종목명, 매수/매도 날짜, 수량, 가격을 직접 입력하고 저장합니다.

투자 전략 기반 조언: 사전에 정의된 다양한 투자 전략(볼린저 밴드, 신고가 돌파 등)을 선택하면, LLM이 매매 기록을 분석하여 해당 전략에 맞는 조언을 제공합니다.

유튜브 기반 전략 생성: 유튜브 영상 URL을 입력하면, LLM이 영상 내용을 분석해 새로운 투자 전략을 추출하고 사용자만의 맞춤형 전략으로 등록합니다.

백테스팅 (추후 개발): 선택한 전략을 과거 주가 데이터에 적용하여 시뮬레이션하고, 객관적인 수익률을 확인합니다.

기술 스택

백엔드: Java (Spring Boot, JPA), PostgreSQL

프론트엔드: Next.js, React

AI: FastAPI, gpt-oss-20b, LangChain

API: KIS Developers API, YouTube Transcript API
